# ChatGPTに依頼するロゴ生成プロンプト

以下をそのままChatGPT（画像生成機能）にコピペしてください。

---

Minecraft mod用のアイコン/ロゴ画像を作ってください。

**Modの内容**: 「Micra Drone」という名前の、プログラムで小さな農業ドローンを操作して畑を自動化するMinecraft mod。プレイヤーはPython風の簡単なコードを書いてドローンに指示を出し、耕す・種を植える・収穫するを自動でやらせる。

**描いてほしいもの**: 小さくて可愛い、青緑色(シアン)のブロック状(voxel)のドローン/ロボットが、畑の上に浮かんでいる構図。周りに小麦・にんじん・かぼちゃなど収穫物を軽く添える。Minecraft本体のブロッキーな世界観に馴染む見た目（滑らかな3DCGではなく、角ばった立方体ベースのスタイル）。

**スタイル**:
- Minecraft公式のアイテム/Modアイコンのような、シンプルでアイコンとして小さく表示されても分かりやすい構図
- 背景はシンプル（単色、または軽いグラデーション。透過は無しでも可）
- 中央にドローンを大きく配置し、余白を作りすぎない
- カラーパレット: シアン/水色のドローン本体、緑の畑、茶色い土、差し色にオレンジ(かぼちゃ)や黄色(小麦)

**フォーマット**: 正方形（1:1）、できれば512x512px以上、PNG形式。

---

## 補足（自分用メモ）

- CurseForgeのプロジェクトアイコンとして使う予定。正方形なら概ねどのサイズでも通るはずだが、大きめ(512px以上)で作っておけば安全。
- 気に入る結果が出るまで、上のプロンプトの「カラーパレット」や「構図」の部分だけ変えて何度か試すとよい。
- ゲーム内のドローンの見た目は、実際にはバニラのAllay(小さな妖精のようなMob)をそのまま流用している(`DroneEntity`はAllayのサブクラス)。似せたい場合は「Allayのような、小さな羽の生えた妖精型ロボット」と付け加えるとイメージが伝わりやすいかもしれない。
